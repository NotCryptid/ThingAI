The Basics

To start you need to create a new chatbot, you can name it whatever you want, then select your desired model and api.

Message Management

Use the Send text block to talk to ThingAI, make sure to set the chatbot name correctly.


Use the Generate with no context block to talk to ThingAI without any chat history.


Use the Attach Image block to give ThingAI an image to analyze.

Chatbot Management

Use the Create chatbot block to create a new instance of ThingAI.


Use the Delete chatbot block to delete an instance of ThingAI.


Use the reset chat history block to delete all the memories of a ThingAI instance.


Use the Chat history as an array block to get the whole chat history of a ThingAI instance.


Use the Import chat history to overwrite the memories of a ThingAI instance.


Use the Import chats from to either merge/update existing chats or delete all chatbots and start over.


Use the All chats as Arrays block to get all the conversations from every ThingAI instance.


Use the Currently active chats block to get the name of every active ThingAI instance.

Image Generation

Use the Generate and get response block to manually generate an image and get its url.


Use the Generate and import as costume block to manually generate an image and import it as a new costume.
